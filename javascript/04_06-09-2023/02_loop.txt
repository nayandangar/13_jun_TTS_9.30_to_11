loop is a count number of iteration.

in javascript there are 6 type of loop 

(1) while loop 
(2) do..while loop 
(3) for loop 
(4) for..in loop (object)
(5) for..of loop (array)
(6) foreach loop (array)

syntax :
while(condition){
    statement;
}

do{
    statement
}while(condition)

for(declaration; condition; increment/decrement){
    statement
}
